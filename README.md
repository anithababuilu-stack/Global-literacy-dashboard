# 📊 Global Literacy & Education Trends Dashboard

This project analyzes global literacy, illiteracy, GDP, and education trends using SQL and Streamlit.

---

## 🚀 Features

- 📈 Literacy trends visualization
- 🌍 Country-wise analysis
- 🧮 SQL query execution
- 📊 Interactive charts using Plotly
- 📉 GDP & schooling insights

---

## 🛠️ Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
